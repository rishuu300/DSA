<h2><a href="https://www.geeksforgeeks.org/problems/implementation-of-deque-using-circular-array--174925/1">Implementation of Deque using circular array</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;"><span style="font-size: 14pt;">Implement a&nbsp;</span><span style="font-size: 18.6667px;"><strong>Deque </strong>(Double Ended Queue)</span><strong style="font-size: 14pt;"> </strong><span style="font-size: 14pt;">using a </span><span style="font-size: 18.6667px;">circular array</span><span style="font-size: 14pt;">, where the size of the array, </span><strong style="font-size: 14pt;">n</strong><span style="font-size: 14pt;">&nbsp;is given.</span><br><span style="font-size: 14pt;">The </span><span style="font-size: 18.6667px;">Deque </span><span style="font-size: 14pt;">must support the following operations:</span></span></p>
<p><span style="font-size: 14pt;"><strong style="font-size: 14pt;">(i)</strong><strong style="font-size: 14pt;">&nbsp;</strong><span style="font-size: 18.6667px;"><strong>insertFront</strong></span><strong style="font-size: 14pt;">(x):</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;">Adds an element x at the front of Deque.<br></span></span><span style="font-size: 14pt;"><strong style="font-size: 14pt;">(ii)</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;"><strong>insertRear</strong></span><strong style="font-size: 14pt;">(x):</strong><span style="font-size: 14pt;">&nbsp;</span></span><span style="font-size: 18.6667px;">Adds an element x at the rear of Deque.</span><br><span style="font-size: 14pt;"><strong style="font-size: 14pt;">(iii)</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;"><strong>deleteFront</strong></span><strong style="font-size: 14pt;">():</strong><span style="font-size: 14pt;">&nbsp;</span></span><span style="font-size: 18.6667px;">Deletes an element from front of Deque.&nbsp;</span><br><span style="font-size: 14pt;"><strong style="font-size: 14pt;">(iv)</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;"><strong>deleteRear</strong></span><strong style="font-size: 14pt;">():</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;">Deletes an element from rear of Deque.</span><br><strong style="font-size: 14pt;">(v)</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;"><strong>frontEle</strong></span><strong style="font-size: 14pt;">():</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;">Gets the front element from queue. Return -1 if it is empty.</span><br><strong style="font-size: 14pt;">(vi)</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;"><strong>rearEle</strong></span><strong style="font-size: 14pt;">():</strong><span style="font-size: 14pt;">&nbsp;</span><span style="font-size: 18.6667px;">Gets the last element from queue. Return -1 if it is empty.</span><br></span></p>
<p><span style="font-size: 14pt;">There will be a sequence of queries&nbsp;<strong>queries[][]</strong>. The queries are represented in numeric form:<br></span></p>
<ul>
<li><span style="font-size: 14pt;">1 x : Call insertFront(x)</span></li>
<li><span style="font-size: 14pt;">2 x : Call insertRear(x)</span></li>
<li><span style="font-size: 14pt;">3 : Call deleteFront()</span></li>
<li><span style="font-size: 14pt;">4 : Call deleteRear()</span></li>
<li><span style="font-size: 14pt;">5 : Call frontEle()</span></li>
<li><span style="font-size: 14pt;">6 : Call rearEle()</span></li>
</ul>
<p><span style="font-size: 14pt;"><span style="font-size: 14pt;">You just have to implement the functions&nbsp;</span><span style="font-size: 18.6667px;"><strong>insertFront</strong></span><strong style="font-size: 14pt;">,&nbsp;</strong><span style="font-size: 18.6667px;"><strong>insertRear</strong></span><strong style="font-size: 14pt;">,&nbsp;</strong><span style="font-size: 18.6667px;"><strong>deleteFront</strong></span><strong style="font-size: 14pt;">, </strong><span style="font-size: 18.6667px;"><strong>deleteRear</strong></span><strong style="font-size: 14pt;">,&nbsp;</strong><span style="font-size: 18.6667px;"><strong>frontEle </strong></span><span style="font-size: 14pt;">and&nbsp;</span><span style="font-size: 18.6667px;"><strong>rearEle</strong></span><strong style="font-size: 14pt;"> </strong><span style="font-size: 14pt;">and the driver code will handle the input &amp; output.</span></span></p>
<p><span style="font-size: 18.6667px;"><strong>Note: </strong>It is guaranteed that all the queries are valid.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 3, q = 6, queries[][] = [[1, 3], [2, 5], [1, 6], [6], [3], [5]]
<strong>Output: </strong>[5, 3]<strong>
Explanation: </strong>Queries on Deque are as follows:<br>insertFront(3): Insert 3 at the front of the Deque.<br>insertRear(5): Insert 5 at the rear of the Deque.<br>insertFront(6): Insert 6 at the front of the Deque.<br>rearEle(): Return the rear element i.e 5.<br>deleteFront(): Remove the front element 6 from the Deque.<br>frontEle(): Return the front element i.e 3.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 2, q = 4, queries[][] = [[2, 4], [3], [6], [5]]
<strong>Output: </strong>[-1, -1]<strong>
Explanation: </strong>Queries on Deque are as follows:<br>insertRear(4): Insert 4 at the rear of the Deque.<br>deleteFront(): Remove the front element 4 from the Deque.<br>rearEle(): As the Deque is empty return -1.</span><br><span style="font-size: 14pt;">frontEle(): As the Deque is empty return -1.</span></pre>
<p><span style="font-size: 14pt;"><strong style="font-size: 14pt;">Constraints:</strong><br><span style="font-size: 14pt;">1&nbsp;</span>≤ n ≤ 10<sup>3</sup><br></span><span style="font-size: 14pt;">1 ≤ number of query ≤ 10<sup>3</sup><br>0 ≤ x<strong>&nbsp;</strong>≤ 10<sup>5</sup></span></p></div>
